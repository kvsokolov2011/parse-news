#Description
**Парсинг новостей сайтов 4geo в portedcheese/site-news пакет новостей**

## Установка
php artisan make:parse-news {--all : Run all}
                      {--menu : Config menu}
                      {--config : Make config}
                      {--controllers : Export controllers}
                      {--jobs : Export jobs}
                      {--scss : Export scss files}
